See https://github.com/tonsky/FiraCode for how to install.

## 3 - Set settings
1. <kbd>Ctrl</kbd> + <kbd>shift</kbd> + <kbd>p</kbd> 
2. Type `settings`
3. Click on: `Preferences: Open settings (JSON)`
4. past the contents of `settings.json` in there
